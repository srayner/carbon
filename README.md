Carbon CMS
==========

A very basic PHP content management system.

History
-------
This project was initially created from the tutorial located here; http://www.elated.com/articles/cms-in-an-afternoon-php-mysql/
The original article was written in 2011 and PHP has come a long way since then.

The intention is to take this simple concept application and update it to use more modern techniques. The following improvements
are to be added;

* directory structure
* object oriented style
* namespaces
* autoloading
* separation of concerns using the MVC pattern

Further changes will include the use of bootstrap for styles.
